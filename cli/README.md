# Usage
<!-- usage -->
```sh-session
$ npm install -g @etherisc/gifcli
$ gifcli COMMAND
running command...
$ gifcli (-v|--version|version)
@etherisc/gifcli/1.1.11-independant linux-x64 node-v10.15.3
$ gifcli --help [COMMAND]
USAGE
  $ gifcli COMMAND
...
```

## `gifcli artifact:send`

Send artifact

```
USAGE
  $ gifcli artifact:send

OPTIONS
  -f, --file=file        (required) truffle artifacts file
  -n, --network=network  (required) network

DESCRIPTION
  ...
  Send artifacts
```

USAGE

  ...

## `gifcli help [COMMAND]`

display help for gifcli

```
USAGE
  $ gifcli help [COMMAND]

ARGUMENTS
  COMMAND  command to show help for

OPTIONS
  --all  see all commands in CLI
```

_See code: [@oclif/plugin-help](https://github.com/oclif/plugin-help/blob/v2.2.3/src/commands/help.ts)_
